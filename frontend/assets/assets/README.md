# Assets Directory

This directory contains static assets for the Civic Issue Reporting & Tracking System frontend.

## Structure

- `images/` - Contains all image assets including icons, logos, and placeholder images
- `icons/` - SVG and PNG icons for different issue types and status indicators
- `data/` - Sample data files for development and testing

## Image Guidelines

- Use compressed images for better performance
- Maintain consistent naming convention: `category-description.extension`
- Preferred formats: PNG for icons, JPG for photos, SVG for scalable graphics
