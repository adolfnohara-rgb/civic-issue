# Icons Directory

Contains icons for the civic issue reporting system.

## Icon Categories

### Status Icons
- `status-pending.svg` - Yellow circle for pending issues
- `status-progress.svg` - Blue circle for in-progress issues  
- `status-resolved.svg` - Green checkmark for resolved issues

### Issue Type Icons
- `issue-pothole.svg` - Road maintenance issues
- `issue-streetlight.svg` - Street lighting problems
- `issue-garbage.svg` - Waste management issues
- `issue-water.svg` - Water supply problems
- `issue-traffic.svg` - Traffic signal issues
- `issue-park.svg` - Park and recreation issues

### UI Icons
- `location-pin.svg` - Location marker
- `camera.svg` - Photo upload
- `user.svg` - User profile
- `admin.svg` - Admin dashboard
- `logout.svg` - Logout button
